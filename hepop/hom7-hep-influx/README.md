HOMER-App, HEPop, TICK Stack
========

## Setup

```bash
docker-compose up
```

to bring up:  

* HEPop Server localhost:9060 (hep-only)
* Homer localhost:9080 (admin/sipcapture) 
* Chronograf localhost:9090 (admin/admin)
  * InfluxDB
  * Kapacitor
  * Telegraf
